## À propos

Portfolio personnel présentant mes compétences en développement web. Un site moderne avec des scènes 3D interactives, des animations fluides et une expérience utilisateur soignée.

## Stack technique

- **React** + **Vite**
- **Three.js** / **React Three Fiber** / **Drei** — rendu 3D
- **GSAP** — animations et scroll
- **Tailwind CSS** — styles et mise en page

## Fonctionnalités

- Modèles 3D interactifs (compétences, héros, contact)
- Animations GSAP au scroll
- Section projets avec liens vers CreaScape, Zentry, Call Me Robyn
- Formulaire de contact (EmailJS)
- Témoignages clients
- Design responsive

## Installation

**Prérequis :** Node.js (v18+), npm

```bash
# Cloner le repo
git clone https://github.com/VOTRE_USERNAME/portfolio_yanis.git
cd portfolio_yanis

# Installer les dépendances
npm install
```

**Variables d'environnement**

Créer un fichier `.env` à la racine pour le formulaire de contact :

```env
VITE_APP_EMAILJS_SERVICE_ID=
VITE_APP_EMAILJS_TEMPLATE_ID=
VITE_APP_EMAILJS_PUBLIC_KEY=
```

Configuration disponible sur [EmailJS](https://www.emailjs.com/).

**Lancer le projet**

```bash
npm run dev
```

Puis ouvrir [http://localhost:5173](http://localhost:5173) dans le navigateur.
